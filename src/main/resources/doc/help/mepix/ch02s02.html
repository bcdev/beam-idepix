2.2. Additional Requirements

To be able to process all options properly, the MEPIX processor needs the following add-ons to be installed in BEAM (if not already done):

  • MERIS L2 Auxdata module
  • GETASSE30 DEM
  • GETASSE30 DEM reader

The MERIS L2 Auxdata and the GETASSE30 DEM reader can be installed with the BEAM module manager. See BEAM help documentation for details (http://www.brockmann-consult.de/beam/doc/help/visat/ModuleManager.html). The GETASSE30 DEM can be downloaded from the BEAM download page. See also BEAM help documentation (http://www.brockmann-consult.de/beam/ doc/help/visat/GETASSE30ElevationModel.html).
