4.6. Control Buttons

From all MEPIX tab panes, the following control buttons can be accessed:
  • 'Run': If this button is clicked, the processor will start the computations.
  • 'Close': If this button is clicked, the MEPIX dialog is closed.
  • 'Help': If this button is clicked, this manual is displayed as online help.
