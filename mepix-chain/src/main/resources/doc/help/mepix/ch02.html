2.1. MEPIX as BEAM plugin

As a BEAM plugin, the MEPIX processor needs the following software to be installed in advance:

  • BEAM, version 4.6 or 4.6.1

The BEAM software can be obtained from the BEAM download page (www.brockmann-consult.de/beam).

The MEPIX software delivery consists of three jar files, the MEPIX main module and adjusted BRR (Bottom of Rayleigh Reflectance) and SDR (Surface Directional Reflectance) modules, which provide additional Java classes used by MEPIX:

  • mepix-chain-1.1.jar
  • beam-meris-brr-2.0.3-SNAPSHOT.jar
  • beam-meris-sdr-2.1-SNAPSHOT.jar

These files need to be copied into the 'modules' folder of the BEAM installation (Figure 2.1). Note that existing versions of these jar files need to be replaced/overwritten. There should not be more than one jar file for each of these modules in the 'modules' folder!
