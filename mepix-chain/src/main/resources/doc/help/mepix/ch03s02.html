3.2. Standard L2 Products (nominal mode)

The standard L2 products which can be derived using the MEPIX tool in nominal, non-QWG mode are also:

  • Cloud Top Pressure
  • Surface Pressure
  • Cloud Flag

In opposite to the QWG mode, the algorithms have been implemented on the basis of the former DPM/IODD versions (ir72a, i7r3A [2]). Overall, the algorithms are quite similar, but there are a few differences in detail (see [1] and [2] for more information).
